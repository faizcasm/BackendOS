# Examples

This directory contains example applications demonstrating how to use BackendOS modules.

## Examples

1. **Basic Server** - Simple server setup with essential modules
2. **Full Featured** - Complete application using all modules
3. **Custom Module** - How to create and integrate your own module

## Running Examples

```bash
# Run an example
npm run example:basic

# Or use ts-node directly
npx ts-node examples/basic-server.ts
```
